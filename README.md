Moderne, animierte Portfolio-Website zur Präsentation meiner Projekte und Skills als Webentwickler. Gebaut mit React und Vite, deployed über GitHub Pages via GitHub Actions. Das Design setzt auf ein dunkles Farbschema mit auffälligen Akzentfarben, 3D-Tilt-Effekten auf Projektkarten, scroll-getriggerten Animationen und einem custom Cursor. Inhalte umfassen eine Hero-Sektion, Projektübersicht, Skill-Darstellung mit animierten Bars sowie eine Über-mich-Sektion.

**Stack:** React 18, Vite, CSS Custom Properties, GitHub Actions
